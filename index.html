PyCalendarGen

Generate customizable calendar pages in PDF format

Download .zip Download .tar.gz View on GitHub

Purpose

PyCalendarGen is a command-line utility that generates calendar pages for use in, for example, a photo calendar. Output is configurable by selecting one of several languages, colors, and fonts. Special dates can be entered into the data file for the corresponding language, so that they will appear in the rendered output.

Installation

Just unpack the archive somewhere useful, and run PyCalendarGen.py from the installation directory (so it will find the fonts).
